
# Configurações iniciais do Ubuntu


## Node 14.19.0 por nvm

```bash 
sudo apt install wget
```
```bash
wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash
```
```bash
source ~/.nvm/nvm.sh
```
```bash
nvm install 14.19.0
```
